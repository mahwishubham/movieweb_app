Welcome to MovieWeb App!

Route Type Description
/add_user POST Use this route to add a user. You can use postman to add user. Required Data: name, email, movies={}
/add_movie POST Use this route to add a movie. You can use postman/ UI to add movie. Required Data: movie name. We use OMDB database to search the movie and add to the list.
/users GET List All Users in Tabular Fashion
/users/<user_id> GET This route will exhibit a specific user’s list of favorite movies.
/users/<user_id>/add_movie POST This route will display a form to add a new movie to a user’s list of favorite movies. You can select the Movie from movie list and click on watch button and enter userId. Or you can use postman with complete movie data or just a movie name.
/users/<user_id>/update_movie/<movie_id> PUT This route will display a form allowing for the updating of details of a specific movie in a user’s list.
/users/<user_id>/delete_movie/<movie_id> DELETE Upon visiting this route, a specific movie will be removed from a user’s favorite movie list.
